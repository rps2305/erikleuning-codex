# Quickstart: Menu Privacy Link and Current Year

## Prerequisites

- A modern browser (Chrome, Firefox, Safari, or Edge).

## Run Locally

1. Open `index.html` directly in a browser.
2. Open `privacyverklaring.html` to verify header and footer navigation.

## Manual Checks

- Top navigation shows no Privacy link on both pages.
- Footer Privacy link remains present and works.
- Footer year matches the current calendar year (YYYY).
- Test with JS disabled: menu and year still correct.
