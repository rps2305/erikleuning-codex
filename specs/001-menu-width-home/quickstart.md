# Quickstart: Menu Home Label and Width Alignment

## Prerequisites

- A modern browser (Chrome, Firefox, Safari, or Edge).

## Run Locally

1. Open `index.html` directly in a browser.
2. Open `privacyverklaring.html` to verify header alignment.

## Manual Checks

- Main menu first label reads "Home" (replacing "Welkom").
- Header/menu and main content share a 1260px max-width on desktop.
- No horizontal scrolling at 320px width.
