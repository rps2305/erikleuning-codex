# Quickstart: Mobile Site Polish

## Prerequisites

- A modern browser (Chrome, Firefox, Safari, or Edge).

## Run Locally

1. Open `index.html` directly in a browser.
2. Open `privacyverklaring.html` to verify footer and privacy navigation.

## Manual Checks

- Viewport checks: 320/360/390/414/768/1024/1440px.
- Keyboard navigation: skip link, menu open/close, cookie banner actions.
- Cookie preferences: analytics toggle saves choice; banner stays closed after choice.
- Text zoom: 200% with no horizontal scroll.
- Images disabled: content remains readable.

## Optional Performance Check

- Run a Lighthouse mobile audit and verify LCP <= 2.5s and CLS <= 0.1.
