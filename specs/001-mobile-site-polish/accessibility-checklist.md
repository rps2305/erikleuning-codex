# Accessibility Checklist

## Keyboard

- [ ] Skip-link visible on focus
- [ ] Mobile menu open/close via Enter/Escape
- [ ] Cookie banner actions reachable by keyboard
- [ ] Focus order follows visual order

## Landmarks + Headings

- [ ] Header/nav/main/footer landmarks present
- [ ] Single H1 with logical H2/H3 nesting

## Contrast + Focus

- [ ] Focus visible on links and buttons
- [ ] Contrast meets WCAG 2.1 AA

## Screen Reader Spot Checks

- [ ] NVDA/VoiceOver reads section headings
- [ ] Cookie banner role + description announced
